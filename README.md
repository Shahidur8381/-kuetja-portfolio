# কুয়েট সাংবাদিক সমিতি (KUET Journalism Association) - Portfolio Website

This is a clean, modern, and responsive portfolio website for **কুয়েট সাংবাদিক সমিতি (KUETJA)**, the official student journalism club of Khulna University of Engineering & Technology (KUET). 
<!--  -->

## 🚀 Project Overview
The website serves as a digital portfolio for the club, showcasing its mission, executive committee members, recent news, and campus activities. It is designed with a journalism-inspired aesthetic, using a minimal color palette (white, black, and maroon) and elegant typography.

### Key Features:
- **Clean & Professional UI:** Minimalist design with a focus on readability.
- **Responsive Layout:** Fully optimized for mobile, tablet, and desktop viewing.
- **Executive Committee:** Detailed listing of the 2025-26 executive body.
- **Featured News:** Recent reports on campus events like Bangla Noboborsho 1433 and administrative changes.
- **Advisory Council:** Information about the club's respected faculty advisors.
- **Interactive Elements:** Smooth scrolling, glowing logo animations, and hover effects.

## 🛠️ Built With
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling with CSS Grid, Flexbox, and Keyframe animations.
- **Google Fonts:** Merriweather & Noto Serif Bengali for high-quality typography.

## 📂 Project Structure
- `index.html`: The main entry point of the website.
- `style.css`: All styling and animations.
- `assets/`: Folder containing logos and news-related images.

## 🏗️ Future Development
> **Note:** This is currently a static portfolio website. Plans for future updates include:
> - Implementing a Backend CMS for dynamic news posting.
> - Advanced gallery/activity management.
> - Integration of a full news portal interface.
